# エレベータピッチ
動画に字幕を付けたい動画視聴者向けの、自動字幕表示を実現するCaptioneeです。
これは動画から音声認識によって字幕を自動で生成することができ、既存の自動字幕表示と違って、高精度な翻訳機能と字幕のカスタマイズ機能が備わっています。
